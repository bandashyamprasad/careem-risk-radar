# Project Summary

Risk Radar is an AI-assisted workflow designed to help Program Managers identify risks, dependencies, blockers, and resource constraints before they impact delivery timelines.

The solution uses a structured prompt to analyze project updates from engineering, product, operations, and external partners, then generates a risk assessment with severity levels, recommended actions, and an executive summary.

The prototype focuses on improving visibility across cross-functional initiatives, highlighting third-party integration risks, delivery blockers, and resourcing concerns.

This approach helps teams proactively manage issues, communicate project health clearly, and support more predictable execution of complex programs.
